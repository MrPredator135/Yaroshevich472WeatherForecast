Ярошевич Олександр 472 Прогноз погоди на тиждень

Проєкт призначений для прогнозування ймовірності опадів

Використовуються дані сервісу [Open-Meteo](https://open-meteo.com/):
Історичні дані: архів за 2 роки (2023-2024) для навчання моделі
Прогнозні дані: поточний прогноз на 7 днів для створення передбачень
Параметри з сайту: температура (max/min), швидкість вітру, сонячна радіація, тривалість сонячного сяйва
Цільова змінна (target): бінарна класифікація (1 — опади будуть, 0 — опадів не буде)

Проєкт автоматично порівнює ефективність трьох алгоритмів машинного навчання:
1. Логістична регресія (Logistic Regression)
2. Випадковий ліс (Random Forest Classifier)
3. Метод k-найближчих сусідів (K-Nearest Neighbors (KNN))
Автоматично обирається модель з найвищою точністю і зберігається для подальших прогнозів

Структура проєкту:
app.py - Web-інтерфейс на базі Streamlit
train.py - Скрипт навчання: завантаження даних, порівняння моделей та збереження кращої
utils.py - Допоміжні функції для роботи з API та обробки даних

Додаткові файли:
requirements.txt - Список необхідних бібліотек для роботи

Спочатку необхідно запустити train.py. Скрипт завантажить дані, навчить три моделі та збереже найкращу у файл weather_model.pkl
Далі необзідно запустити інтерфейс командою streamlit run app.py
В інтерфейсі натисніть кнопку "Get 7-day Forecast". Додаток завантажить свіжий прогноз, обробить його через збережену модель та виведе ймовірність дощу
